Leetcode Problem 704 - Binary Search

Problem

Given a sorted (in ascending order) integer array nums of $n$ elements and a target value, write a function to search target in nums. If target exists, then return its index, otherwise return -1.

Cost function of Logistic regression: $$ J(\theta) = \frac{1}{m} \sum_{i=1}^m f^{(i)} + \frac{\lambda}{2m} \sum_{j=1}^n \theta_j^2 $$ where \( f^{(i)} = -y^{(i)} \log (h_\theta(x^{(i)})) - (1-y^{(i)}) \log(1 - h_\theta(x^{(i)})) \) and hypothese function $h_\theta(x^{(i)}) = g(z_\theta(x^{(i)})) = \frac{1}{1 + e^{-z_\theta(x^{(i)})}}$. 


    class Solution {
        public int search(int[] nums, int target) {
            int left = 0;
            int right = nums.length - 1; // -1 for zero-based indexing
            
            while (left <= right){
                int middle = left + (right -left)/2;
                
                if (nums[middle] < target){
                    left = middle + 1;
                }
                else if (nums[middle] > target){
                    right = middle - 1;
                }
                else{
                    return middle;
                }                
            }
            return -1;
        }
    }
